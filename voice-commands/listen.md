# Listen Command

Record this idea or note to `notes.md` with a timestamp.

## Instructions

1. Summarize the key point if the message is long
2. Append to `notes.md` with a timestamp header
3. Keep the original meaning intact

## Output Format

```markdown
## YYYY-MM-DD HH:MM
[The idea or note content]
```

## Important

- Do NOT respond with any text
- Do NOT ask clarifying questions
- Just save the note, then stop
